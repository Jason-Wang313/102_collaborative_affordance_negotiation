# Hostile Prior Work

The hostile set contains 100 papers and the manuscript bibliography now draws 230 entries from the local deep-read pool. The strongest threat categories remain:

- Physical human-robot collaboration surveys.
- Shared autonomy and POMDP goal inference.
- Human-model MPC and inverse-RL intent models.
- Affordance-based HRC planners.
- Capability-map TAMP.
- Active clarification and legible communication.
- Risk-aware and conformal safety filters.
- Ergonomics, burden, trust, and human-factor models.
- Robot collaborative manipulation systems with deployment claims.

The novelty boundary is not "shared autonomy plus more metrics" or "ask the human more often." The contribution must show that risk-calibrated negotiation changes what the robot represents and which collaborative actions it can safely choose.

Current v5 local evidence supports that boundary against executable proxy baselines. It does not settle the external prior-work threat because no real human-robot study, accepted high-fidelity benchmark, calibrated external human-intent log, or trained checkpoint evidence is present.
